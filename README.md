# 🛡️ SOC Analyst Labs & Investigations 🛡️

Welcome to my SOC Analyst lab repository, where I document hands-on cybersecurity investigations, incident response scenarios, and network security analysis performed within controlled lab environments.

This repository focuses on developing practical blue team and SOC Level 1 skills through real-world style investigations involving log analysis, threat detection, lateral movement identification, data exfiltration analysis, and attacker behaviour tracking.

The labs are designed to strengthen analytical thinking, improve understanding of attack chains, and build experience using common security tools and methodologies used by SOC analysts and incident responders.
